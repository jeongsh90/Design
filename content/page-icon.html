시스템 전역에서 일관된 시각 언어를 유지하기 위해 모든 UI 컴포넌트는 동일한 아이콘 규칙을 사용하며 CSS 변수 기반의 SVG 마스크(Mask-image) 방식으로 제공됩니다.

시스템 아이콘

시스템 아이콘은 인터페이스 전반에서 일관된 시각 언어를 유지하도록 설계된 기본 아이콘 세트입니다. 각 아이콘은 동일한 규칙(선 두께, 여백, 비율)을 기반으로 제작되어 다양한 UI 요소와 결합했을 때 시각적 안정성과 기능적 명확성을 제공합니다. 또한 아이콘 클릭 시 해당 아이콘의 코드가 자동으로 클립보드에 복사되어, 개발 및 디자인 과정에서 빠르게 활용할 수 있습니다.

코드 사용 예시

아래는 시스템 아이콘을 HTML 코드 형태로 사용할 때의 예시입니다. 아이콘은 모두 <i class="icon ..."></i> 구조로 호출됩니다.
클래스 조합에 따라 원하는 아이콘을 지정할 수 있습니다. 


                <i class="icon arrow-left"></i> 
                <i class="icon arrow-up"></i>
                <i class="icon plus"></i>

아이콘 사이즈 규칙

아이콘은 폰트 아이콘(Font Icon) 방식으로 제작되었으며, 크기는 font-size, 색상은 color 속성을 통해 지정합니다. 기본 정렬 기준은 1em이며, 텍스트 라인과 자연스럽게 맞추기 위해 -0.125em의 하단 보정값이 적용됩니다. 


                .icon {
                    font-size: 20px;
                    color: #4e8cff;
                }
                

                <i class="icon arrow-left" style="font-size:20px; color:#4e8cff;"></i>
